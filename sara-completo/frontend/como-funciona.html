Funcionamento

Como o S.A.R.A funciona

Do registro na fábrica até a verificação pelo consumidor — cada produto tem uma identidade única e rastreável.

01
Fabricação

Registro do produto

Durante a fabricação ou distribuição, cada produto recebe um QR Code único gerado pelo sistema S.A.R.A e registrado no banco de dados central.

QR Code único Nome do produto Lote de fabricação Código de barras Data de fabricação Local de distribuição
02
Ponto de venda

Registro da venda

Quando o produto é vendido, o código de barras é escaneado normalmente no caixa. O S.A.R.A registra automaticamente a venda com data, hora e local.

Venda registrada Data e horário Local da venda ID do produto
03
Consumidor

Verificação via celular

Após a compra, o consumidor escaneia o QR Code com a câmera do celular. O sistema abre automaticamente uma página com o resultado da verificação.

Câmera do celular Sem app necessário Resultado imediato
04
Resultado

Autêntico ou alerta de fraude

O sistema exibe as informações do produto se ele for autêntico, ou emite um alerta de possível falsificação caso o QR Code já tenha sido escaneado anteriormente.
