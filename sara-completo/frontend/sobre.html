Sobre o S.A.R.A

Projeto acadêmico desenvolvido para combater a falsificação de produtos por meio de tecnologia acessível e rastreamento inteligente.

O S.A.R.A — Sistema Automático de Reconhecimento de Autenticidade nasceu da necessidade de criar uma solução tecnológica capaz de proteger consumidores e empresas contra a falsificação de produtos em larga escala.

O sistema propõe a integração de QR Codes únicos gerados no momento da fabricação, vinculados a um banco de dados central que registra toda a cadeia de distribuição e venda de cada produto. Com isso, qualquer consumidor pode verificar a autenticidade de um item apenas escaneando o código com a câmera do celular — sem precisar instalar nenhum aplicativo.

O projeto foi desenvolvido como trabalho acadêmico e contempla arquitetura de software completa: banco de dados relacional, API RESTful, front-end responsivo e painel administrativo.
